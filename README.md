# 酒店管理系统 java425

本项目是基于Spring Boot和Vue的前后端分离架构的酒店管理系统。

## 技术栈

- 数据库：MySQL
- 开发语言：Java
- 前端技术：Vue、Element-UI
- 后端技术：Spring Boot、MyBatis Plus

## 功能模块

### 系统角色

系统定义了两个主要角色：管理员和用户。

#### 管理员

- 登录功能
- 用户管理
  - 处理酒店预约订单
  - 查看订单列表
- 房间管理
  - 房间列表
  - 房间详情查询
  - 添加房间
  - 修改房间信息
- 评论列表管理

#### 用户

- 注册功能
- 登录功能
- 查询空余房间
- 房间预约
- 我的订单
  - 查看预约订单

## 运行环境

系统要求以下运行环境：

- 后端：Java 8 或以上版本，MySQL 5.7 或以上版本
- 前端：现代浏览器环境

## 部署步骤

1. 准备所需的环境，包括Java和MySQL服务。
2. 创建数据库，导入项目提供的SQL文件。
3. 部署后端服务，配置数据库连接。
4. 部署前端项目，确保与后端服务通信正常。
